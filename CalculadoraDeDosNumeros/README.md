# CalculadoraDeDosNumeros

El proyecto consiste de una calculadora que hace operaciones matemáticas con dos números.

### Características
- Realiza suma, resta, multiplicación y división.
- Muestra mensaje de error si se intenta dividir entre 0.
- Solo permite ingresar números.

> :exclamation:[IMPORTANTE] Este proyecto se me fue propuesto como una actividad para aplicar mi conocimientos con el lenguaje de C#.
