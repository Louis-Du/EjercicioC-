# Calculadora de Promedio de Notas

Simple calculadora para calcular el promedio de 4 notas con procentajes de 30%, 40%, 20%, 10%.

Caracteristicas
- Muestra mensaje de error si se ingresa una letra.
- Permite el ingreso de digítos como puntos y comas.
- Tiene una interfaz con un buen diseño con botones de salida, limpiar las cajas de texto y calcular el promedio.
